📖 About the Project

SmartSchedAI is a web-based AI-powered scheduling system designed to automate timetable generation and scheduling efficiently.
It helps avoid conflicts in faculty allocation, room assignment, and exam scheduling.

🚀 Features
• Automated timetable generation
• Conflict-free scheduling logic
• Clean and responsive UI
• Real-time backend processing
• Deployed on cloud (Render)

🛠 Tech Stack
• Python
• Flask
• Flask-SocketIO
• HTML
• CSS
• Gunicorn
• Render (Cloud Deployment)
